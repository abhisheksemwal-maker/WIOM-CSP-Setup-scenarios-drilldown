# INS-1047 — SCHEDULED (install due soon / slot day)

## Scenario
A confirmed slot is scheduled for **today**. CSP needs to begin work at any point in the day. State is `SCHEDULED` with `isSlotDay = true` triggering the "today" banner variant.

## Banner (isSlotDay = true)

|---|---|
| Icon | `Schedule` |
| Title | `आज सेटअप का दिन है` |
| Subtitle | `किसी भी वक्त शुरू कर सकते हो` (oblique case — was `कोई भी वक्त`) |
| Accent | `stateWarning` |
| Bg | `bgUrgent` |

## Banner (isSlotDay = false — future scheduled)
| Field | Value |
|---|---|

| Bg | `bgPositive` |

## Hindi rule
`किसी भी वक्त` uses oblique case for time expressions. `कोई भी वक्त` is grammatically wrong (nominative) in this context.
