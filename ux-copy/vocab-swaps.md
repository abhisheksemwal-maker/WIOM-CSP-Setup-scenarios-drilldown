# Vocabulary Swaps — Quick Reference

Short lookup table for copy reviewers. Full rationale lives in [`hindi-anchors.md`](hindi-anchors.md).

## Domain nouns

| Before | After |
|---|---|
| ग्राहक | कनेक्शन |
| व्यक्ति (as role) | टेक्निशियन |
| डिवाइस | नेटबॉक्स |
| कैरी फी | पड़े नेटबॉक्स का किराया |
| फीस (for carry) | किराया |
| बिल | बिल (kept — widely understood) |

## Transliteration (Latin mid-string → Devanagari)

| Before | After |
|---|---|
| `सेटअप submit हुआ` | `सेटअप सबमिट हुआ` |
| `verify शुरू हुआ` | `वेरिफाई शुरू हुआ` |
| `कनेक्शन ने स्लॉट confirm नहीं किया` | `कनेक्शन ने स्लॉट कन्फर्म नहीं किया` *(pending)* |
| `पुराने स्लॉट cancel हुए` | `पुराने स्लॉट कैंसल हुए` *(pending)* |

## Matra fixes

| Wrong | Right |
|---|---|
| वेरीफाई | वेरिफाई |
| वेरीफिकेशन | वेरिफिकेशन |

## Plural / honorific

| Wrong | Right |
|---|---|
| काम कर रहा है (for a named executor) | काम कर रहे हैं |
| काम कर रही है (feminine, for a named executor) | काम कर रही हैं |
| Annu काम कर रहा है | Annu काम कर रहे हैं |

## Time / case

| Wrong | Right |
|---|---|
| कोई भी वक्त | किसी भी वक्त |
| कोई भी समय | किसी भी समय |

## Action cues (replace vague status)

| Status-style | Action-style |
|---|---|
| अभी जवाब बाकी है | समय चुनना बाकी है |
| पिछला स्लॉट चूक गए | कनेक्शन के लिए नया समय चुनें |
| देरी हो गई | {name} ने {thing} में देरी कर दी |
| तैयार नहीं | {name} ने अभी शुरू नहीं किया |

| Sanskritised | Native |

| विवरण | डिटेल |
